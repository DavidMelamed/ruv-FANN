# ruv-marketing-swarm

This workspace hosts experimental marketing automation tools built in Rust. Components include:

- **fann_core** – re-export of the upstream neural network library.
- **marketing_divergent** – common logic and WASM agents.
- **marketing_swarm** – gRPC + REST API exposing marketing entities with autogenerated OpenAPI spec.
- **Swarm Studio** – Tauri desktop app in `apps/swarm-studio`.

Scripts and Docker compose files are provided for local setup. Documentation lives in `docs/` and can be built with `mdbook`.
